## How to run

**Web**
```bash
# Enter the web folder with
$ cd NLW_02-Poffy/web

# Install the dependencies
$ npm install

# Run the application
$ npm start
```


**Server**
```bash
# Enter the server folder with
$ cd NLW_02-Poffy/server

# Install the dependencies
$ npm install

# Run the server
$ npm start
```


**Mobile**
```bash
# Enter the mobile folder with
$ cd NLW_02-Poffy/mobile

# Install the dependencies
$ npm install

# Run the app
$ npm start

# Scan the QR Code displayed in the browser window
```
